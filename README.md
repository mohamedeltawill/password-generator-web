# 🔐 Streamlit Password Generator

A secure, web-based password generator built with Python and Streamlit.

## 🚀 Features

- Set custom password length
- Choose to include:
  - Uppercase Letters
  - Lowercase Letters
  - Numbers
  - Symbols
- One-click password generation

## 🛠️ Technologies Used

- Python 3
- Streamlit

## 🧪 How to Run

```bash
pip install streamlit
streamlit run app.py
```
